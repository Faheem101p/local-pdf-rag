## Features

- Upload PDF documents
- Semantic search using FAISS
- Local LLM inference with Ollama
- Conversation memory
- Source page citations
- Streamlit web interface

## Tech Stack

- Python
- Streamlit
- FAISS
- Sentence Transformers
- Ollama
- PyPDF

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
